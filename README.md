# Cuboid Insertion Checker

Determine whether a rigid rectangular cuboid can be inserted through a rectangular
opening into a hollow enclosure, using a 6-DOF collision-aware path solver
(direct fit -> tilt sweep -> RRT-Connect) rendered in an interactive Three.js viewport.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

Deployed with Vercel.
